2. Click "Visual Editor" or "Admin" in the footer
3. Start editing!

## Visual Editor - Quick Guide

### What You Can Edit
- **Hero**: Headline, title, description, CTA button
- **Features**: Title and description
- **Booking**: Form title, description, company name
- **Styling**: Colors and fonts

### How to Use
1. Click a section button (Hero, Features, Booking, Styling)
2. Edit fields - **see changes instantly in preview**
3. Click "Save" when done
4. That's it!

### Pro Tips
- Changes appear in real-time - no refresh needed!
- Toggle preview with the eye icon
- Use color pickers for easy color selection
- Switch sections quickly with the buttons

## Admin Panel - Quick Guide

### Content Tab
- **Hero Section**: Main headline, title, description, button text
- **Features Section**: Title and description
- **Booking Form**: Title and description
- **Footer**: Company name and tagline

### Styling Tab
- **Colors**: Primary and accent colors with live preview
- **Typography**: Heading and body fonts with preview

### How to Use
1. Switch between "Content" and "Styling" tabs
2. Edit any field or select colors/fonts
3. Click "Save Changes" button
4. Click "Preview" to see changes in new tab

## Tips for Both Editors
- The headline should have one word per line
- The word "APPOINTMENT" will be highlighted in your primary color
- Use hex codes (#RRGGBB) for colors
- Font changes apply site-wide
- Save frequently!
